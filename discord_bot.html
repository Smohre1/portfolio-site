Discord Bot

This is a private passion project that I work on in my free time. A multi-feature Discord bot built in Python using discord.py that includes interactive games, moderation utilities, automated messaging tools, and a Secret Santa coordination system. The project focuses on command handling architecture, persistent data storage using text-based databases, and building interactive server utilities for real-world community use.

Bot Actions

This section showcases some of the bot's commands and functions.

Send Messages

Random Jokes and Stories,
8 Ball Fortune,
Gifs,
Roll a Die (d4 - d%)

Currently Exploring

Today's Holidays

Mod Controls

Create Polls,
Delete all messages in a channel (used in birthday planning or christmas channel to clear for next time)

Secret Santa

Allows users to DM the bot to submit three ideas for gifts,
Stores responses in structured text files,
Stores everyone's names (the ones who joined) into a separate text file,
Has a mod command that sends everyone their secret santa (does not allow a user to get themselves)

Games

Wordle,
Blackjack,
Trivia questions

Currently Exploring

Bingo, Other Smaller Game Ideas

Server Economy

Games give coins,
Stores amount of coins each user has in a .txt file,
Can add and subtract coins from a specific user

Technical Details

Built using Python and discord.py with modular command handlers,
Data persistence handled through structured text storage,
Implements user-state tracking, randomized assignment logic,
Permission-restricted moderator commands
