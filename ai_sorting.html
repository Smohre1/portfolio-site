AI Sorting Game

This capstone project combines my Artificial Intelligence and Game Design track experience into a single Unity-based reinforcement learning system using ML-Agents. I developed a ball sorting environment and trained an AI agent to classify objects by color using reward-based learning. Training began with a single object class and progressively scaled to multiple classes as performance stabilized. The project demonstrates environment design for reinforcement learning, curriculum-style training progression, and integration of machine learning agents into interactive gameplay systems.

Technical Details

Built in Unity using ML-Agents and C#,
Designed a reinforcement learning environment with reward shaping and progressive difficulty scaling,
Implemented curriculum-style training by increasing object variety after convergence at earlier stages.

Skills Demonstrated

Reinforcement learning environment design,
Curriculum training strategy,
Unity ML-Agents integration,
Agent reward shaping,
Simulation-based AI training workflows
